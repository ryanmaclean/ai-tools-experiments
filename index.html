Jason's List of Tools To Explore

1. AI-Powered Coding Assistants & Developer Tools

  • Claude Code - Anthropic's latest AI-powered code editor (from a CLI app)
  • OpenRouter – API routing for AI models, allowing access to various AI models via a single interface.
  • Web Dev Arena – AI-powered development environment.
  • LMArena.ai – AI benchmarking and development tool.
  • Tabnine – AI-powered autocomplete for software development.
  • Codeium – AI-driven coding assistant with free access for developers.
  • Replit Ghostwriter – AI coding assistant built into the Replit platform.
  • CodiumAI – AI for generating test cases and improving code quality.

2. AI Model Benchmarking & Optimization

  • LlamaBench – A benchmarking tool for evaluating Llama-based AI models.
  • LiveBench.ai – Real-time AI model performance benchmarking.

3. AI Workflow Automation & Agents

  • FloWise.AI – No-code/low-code tool for AI workflow automation.
  • MindPal AI – AI-powered automation similar to Zapier, facilitating AI agent workflows.
  • Jina.ai – AI-powered search and multimodal application framework.
  • AgentGPT – Deploy and run autonomous AI agents in a browser.

4. AI-Powered Research & Knowledge Management

  • Elicit – AI-powered research assistant for literature review and data extraction.
  • MyMemo – AI-enhanced note-taking and memory management tool.
  • Napkin.ai – AI-assisted tool for organizing ideas and creative thought processes.
  • SciSpace – AI-powered research assistant for academic papers and literature review.
  • Consensus – AI search engine that finds research-backed answers.
  • Semantic Scholar – AI-driven search and recommendation engine for academic research.

5. AI Chatbots & Conversational Models

  • Claude – AI chatbot developed by Anthropic.
  • Abacus.ai – AI platform for enterprise-level applications, including conversational AI and predictive analytics.
  • Pi AI – AI chatbot designed for friendly and engaging conversations.
  • YouChat – AI conversational assistant with web search capabilities.

6. AI Content & Detection Tools

  • Zerogpt.com – AI-generated content detection.
  • Loudly – AI-powered music generation tool, with genres including Lo-Fi and Bluegrass.
  • GPTZero – AI detection tool for identifying AI-generated content.
  • DeepAI Text Generator – AI-powered text generation tool.
  • Runway ML – AI-powered video and content generation platform.
  • Boomy – AI-generated music creation tool, enabling users to create and release songs.

7. AI Tools for SRE & DevOps Engineers

  • Monitaur – AI-powered reliability and compliance monitoring for machine learning in production.
